A fullstack app using Google Maps API and Petfinder's API

Foodie Foodie Truck Truck

A foodtruck finder app that shows users their location and nearby foodtrucks.

Utilizing Google Maps API, Foodie Foodie Truck Truck allows users to see foodtrucks near them as well as get infomation about the food truck (e.g. contact info, description, hours, etc.) and then chose to get directions to the locaiton of the food truck.

This app has been designed for mobile and not yet formatted for standard browser views. I recommend viewing this on a mobile device for the best experience!

Whack-A-Trump

A frontend web game

Whack-A-Trump is a modern twist on the classic Whack-A-Mole game. Built with only HTML, CSS and Javascript the game allows two players to compete against the clock see who can get the most 'whacks'.
